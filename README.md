# PI Social Media
MERN stack

# Tech
## Frontend : 
    ReactJs, Redux
## Backend : 
    NodeJS, ExpressJS
## Database:
    MongoDB, Mongoose

# Features:
1. registration page & Login page
2. users can upload their images for their profiles
3. Post CRUD
4. Like & Dislike Post, Comment
5. Add Friends 
6. Dark mode
